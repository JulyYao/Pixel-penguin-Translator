# Public Release Checklist

This directory is the clean open-source release of Pixel Penguin.

Before publishing:

- Confirm no API keys are present.
- Confirm no personal dialogue records are present.
- Confirm `miniprogram/`, `android-app/`, `对话记录/`, and private workspace files are not included.
- Add a `LICENSE` file if you want a formal open-source license.
- Create a fresh GitHub repository and upload the contents of this directory.

Recommended deploy root: this directory itself.
