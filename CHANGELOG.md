# 更新日志

## v1.2 - 2026-07-07

- 大幅度优化代码结构：拆分配置、工具函数和语言识别模块，降低 `app.js` 维护压力。
- 添加归档文件夹：归档项目统一收纳到项目栏里的“归档文件夹”，不再混入普通最近列表。
- 添加快速交换翻译语言：输入区的语言方向可以点击临时互换，本次输入生效，发送后恢复自动识别。
- 支持左右气泡方向快速修正：可以通过项目语言互换和本次翻译方向互换，修正语言方向识别错误带来的气泡位置问题。
- 添加界面语言选择：默认英语，支持简体中文、日语、韩语、法语、德语、西班牙语、葡萄牙语、俄语和阿拉伯语等主流 UI 语言。
- 翻译语言名称会跟随界面语言本地化显示，例如英文界面下显示 `Chinese ⇄ French`。
- 优化多语言菜单按钮：根据文字长度自动调整字号，尽量保持菜单单行显示，减少换行和溢出。
- API Key、备用 API Key、Gist Token 不再以明文保存到本地存储；旧明文设置会在读取后自动迁移为密文结构。
- 添加英文版 README：新增 `README_EN.md`，便于英语用户了解项目功能和部署方式。
